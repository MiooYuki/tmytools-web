  • 报名视频批量下载

图文说明（下面有视频教程）

  1. 获取Json文本。

    以海盐为例，登录海盐的目宝，找到相应的报名活动，点击报名记录。



    按F12打开开发者工具，依次点击 网络（Network），Fetch/XHR。



    筛选通过的记录，右侧就会出现当次请求的记录。注意这里用作演示，下面就只选择每页显示10条信息，所以返回的数据也是10条。



    点击这条请求记录，点击响应（Response），Ctrl + A（Mac Command + A）全选复制。



    粘贴进输入框。



  2. 获取各列的 Key。

    除了用户昵称，头像，创建日期，其他项都是由用户自定义创建的，这些自定义项的 Key 都是动态生成的，不是固定的，所以需要手动指定这些项的 Key。

    回到活动页，依次展开 data > list > 0：



    可以看到该活动的四个自定义项的 Key：signnew_486_0，signnew_486_1，signnew_486_2，和 signnew_486_4。

    通过右边的值可以判断出分别对应 真实姓名，手机号，身份证号，上传的视频。

    将 signnew_486_0 复制到 姓名列 Key，signnew_486_1 复制到 手机号列 Key，signnew_486_4 复制到 媒体列 Key。

    姓名列 Key 和 手机号列 Key 用于下载的文件的命名，不填也没关系，只不过下载出来的文件名就不带姓名和手机号了。媒体列 Key 必须指定。



    点击提交任务，会获得当次任务的 RequestKey，用于任务执行完成之后下载本次任务的压缩包，请保存好。



    提交任务后，后台会开始进行视频的压缩打包任务。只需耐心等待任务执行完成即可。一般10条视频需要等待10分钟左右。

  3. 下载压缩包。

    大致感觉任务跑完了，可以尝试点击下载压缩包按钮，如果任务执行完成，即可下载压缩包。



    如果任务没有执行完，会提示 “任务未完成”。



    如果压缩包下载完成，请点击删除文件按钮，删除掉存在服务器（我电脑）上的文件。
